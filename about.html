About My Journey

Who I Am

I'm John Olawoye, a dedicated frontend developer with a passion for creating intuitive and efficient user interfaces. My journey into tech started with curiosity about how websites work, and it has evolved into a commitment to building accessible, user-centered web experiences using React, TypeScript, and modern web technologies.

Areas I'm Working to Improve

While I'm confident in my HTML, CSS, and JavaScript fundamentals, I recognize there are areas where I need to grow. Backend integration and API design patterns are still challenging for me, and I sometimes struggle with complex state management in larger applications. I'm also working on improving my testing skills and understanding performance optimization techniques. This internship is the perfect opportunity to tackle these areas head-on.
